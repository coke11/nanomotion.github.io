NanoOS

NanoOS is a simple, yet capable and powerful OS for everyone.
It's also free, and open-source.






Features

NanoOS comes packed full of features. Here are just a few:
- You can return to Windows at any time
- App hub with built-in web browser and developer console
- Build your own applications using the NanoOS application template for Visual Basic
- Update check, so you can always get the latest version






System Requirements

- Windows XP or above (7 or 10 recommended)
- 2GB of RAM or more (6GB+ recommended)


Developer Requirements

- Visual Studio 2015 or higher (VS2015 recommended)
- Visual Basic or C# package installed on Visual Studio (VB recommended)
